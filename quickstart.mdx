## Démarrage en trois étapes

Configurez votre environnement local et commencez à contribuer à la documentation.

### Étape 1 : Configuration de l'environnement local

<AccordionGroup>
  <Accordion icon="copy" title="Cloner le dépôt">
    Clonez le dépôt de documentation localement pour pouvoir faire des modifications et les prévisualiser.
    
    ```bash
    git clone <repository-url>
    cd docs
    ```
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Démarrer le serveur de prévisualisation">
    1. Installez le CLI Mintlify : `npm i -g mint`
    2. Naviguez vers le répertoire de documentation et exécutez : `mint dev`
    3. Ouvrez `http://localhost:3000` pour voir la documentation en direct !
    
    <Tip>La prévisualisation se met à jour automatiquement lorsque vous modifiez les fichiers.</Tip>
  </Accordion>
</AccordionGroup>

### Étape 2 : Explorer la documentation

<Accordion icon="book" title="Découvrir le template de feature">
  Consultez le [template de développement de feature](/feature/template) pour comprendre notre méthodologie de développement.
</Accordion>

### Étape 3 : Contribuer

<Accordion icon="rocket" title="Publier vos modifications">
  1. Faites vos modifications
  2. Committez et poussez vos changements
  3. La documentation sera mise à jour automatiquement !
</Accordion>
